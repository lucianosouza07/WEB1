Sobre o CineCatálogo

O CineCatálogo é um projeto de estudo desenvolvido para aplicar os conceitos de CSS aprendidos no curso de Desenvolvimento Web I. Aqui, exploramos a estilização de cores, fontes, bordas, espaçamentos, e o uso de display flex para criar layouts responsivos e agradáveis. Todo o conteúdo é fictício e serve apenas para fins educacionais.
